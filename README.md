## Troubleshooting

**The extension does not detect any media on the page.**
Make sure you are on a supported Perfect Girls video page with the `/video/<id>/` route pattern and that the embedded player has fully loaded.

**The download fails or stops midway.**
Check your internet connection and ensure the stream source is still active. Some streams may have expiration windows.

**The popup does not appear when I click the icon.**
Try refreshing the page and clicking the icon again. If the issue persists, reinstall the extension from the latest release.

**I see an error about unsupported formats.**
The extension currently supports m3u8 and mp4 streams. If the page uses a different format, it may not be detectable.

**The download starts but the file is corrupted.**
This can happen if the stream timed out during capture. Try the download again from the beginning.
